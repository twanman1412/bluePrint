
### Core Files
- `commonAST.h` - Common types, type system, and shared utilities
- `programAST.h` - Top-level program structure

### Language Construct Files
- `blueprintAST.h` - Blueprint declarations and contracts
- `classAST.h` - Class declarations and implementations
- `enumAST.h` - Enum declarations
- `exprAST.h` - Expressions (values)
- `stmtAST.h` - Statements (actions and control flow)
- `importAST.h` - Import declarations
- `exportAST.h` - Export declarations
